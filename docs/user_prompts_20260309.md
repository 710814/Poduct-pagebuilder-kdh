## 프롬프트 1: 컬러소개/코디 섹션의 프롬프트에 모든 색상이 포함되는 문제
- **내용**: 컬러소개 섹션과 코디컷 섹션에서 모든 색상의 프롬프트가 한꺼번에 표시되고 있어 토큰 낭비 우려
- **조치**: `replaceColorPlaceholders` 함수에서 실제 사용된 색상만 Tier 1/2 블록 생성하도록 수정

## 프롬프트 2: 컬러별 코디 섹션 후면 이미지가 다른 컬러로 표시되는 문제

- **히스토리**: 이전에 해결되었던 이슈가 최근 기능 수정 중 재발한 것으로 추정
- **조치**: 원인 분석 진행 중
