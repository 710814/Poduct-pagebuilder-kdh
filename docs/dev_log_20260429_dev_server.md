# 개발 로그 (2026-04-29)

## 작업 내역
- **작업 내용**: Vite 개발 서버를 3000번 포트로 구동
- **세부 사항**:
  - `package.json`의 `scripts` 확인 결과, 기본 `dev` 명령어는 `vite`로 설정되어 있음.
  - 사용자의 요청에 따라 3000번 포트로 구동하기 위해 `npm run dev -- --port 3000` 명령어를 실행함.
  - 초기 실행 시 3000번 포트가 이미 사용 중이어서 3001번 포트로 자동 할당됨.
  - 3000번 포트를 점유하고 있던 기존 프로세스(PID: 42267)를 확인 후 `kill -9` 명령어로 종료 처리함.
  - 이후 다시 `npm run dev -- --port 3000`을 실행하여 성공적으로 3000번 포트에서 서버를 구동함.
  
## 버그 및 트러블슈팅 내역
- **이슈**: 3000번 포트 충돌 (`Port 3000 is in use, trying another one...`)
- **원인**: 기존에 실행 중이던 node 프로세스가 3000번 포트를 점유하고 있었음.
- **해결**: `lsof -i :3000`으로 PID를 확인하고 프로세스를 강제 종료한 뒤 서버를 재시작함.
